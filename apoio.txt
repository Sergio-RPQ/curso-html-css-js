Atributos para formulário
Atributo		Descrição
accept-charset	Especificar o conjunto de caracteres no envio do formulário (default: charset da página).
action			Especificar a url que receberá os dados.
autocomplete	Especificar se o browser autocompletará o formulário (default: on).
enctype			Especificar o encoding na submissão dos dados (default: url-encoded).
method			Especificar o método de envio HTTP dos dados (default: GET).
name			Especificar um nome para identificar o formulário (DOM: document.forms.name).
novalidate		Especificar que o browser não deve validar o formulário.
target			especificar o target para o envio (default: _self).
